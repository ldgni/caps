# Caps

> E-commerce shop selling limited edition keyboards

## Running Locally

1. Clone the repository

```sh
git clone https://github.com/ldgni/caps.io
```

2. Navigate inside the repository

```sh
cd repo-name
```

3. Install dependencies

```sh
npm install
```

4.  Start the development server

```sh
npm run dev
```
